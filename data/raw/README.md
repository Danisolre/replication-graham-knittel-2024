# data/raw

Aquí se almacenarán los datos originales del repositorio de replicación:

- **ECF_final.csv** — Indicador de huella de carbono laboral (overall ECF) por condado. Fuente: [MIT-CEEPR/ecf](https://github.com/MIT-CEEPR/ecf)
- **Shapefile de condados** — Geometría de condados de EE.UU. Fuente: U.S. Census Bureau (TIGER/Line)

> **Nota:** Los archivos de datos no se incluyen en el repositorio por su tamaño. Seguir las instrucciones del README principal para descargarlos.
